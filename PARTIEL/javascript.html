Introduction à JavaScript

Que peut faire JavaScript ?

JavaScript est le langage de programmation du web.

Il peut calculer, manipuler et valider les données.

Il peut mettre à jour et modifier à la fois HTML et CSS.

JavaScript Peut Modifier Le Contenu HTML

L'une des nombreuses méthodes HTML JavaScript est getElementById().

L'exemple ci-dessous "trouve" un élément HTML (avec id="demo"), et change le contenu de l'élément (innerHTML) en "Hello JavaScript":

Exemple

document.getElementById("demo").innerHTML = "Hello JavaScript";
